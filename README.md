# Glipearte - Website Institucional

## 📋 Descrição do Projeto

O **Glipearte** é uma plataforma de locação de artigos para festas com o conceito inovador "Pegue e Monte". O site institucional foi desenvolvido para estabelecer uma presença digital profissional que reflita a alegria e sofisticação dos eventos que a empresa ajuda a realizar.

## 🎯 Objetivo

O principal objetivo é servir como uma ferramenta central de negócios, oferecendo:
- Apresentação profissional dos serviços
- Catálogo de produtos disponíveis para locação
- Sistema de orçamento simplificado
- Informações de contato e localização
- Experiência de usuário intuitiva e alegre

## 🌟 Funcionalidades Implementadas

### 1. **Design Responsivo**
- Layout adaptável para desktop, tablet e mobile
- Otimização para diferentes tamanhos de tela
- Suporte a alta resolução (Retina)
- Modo escuro opcional

### 2. **Navegação Interativa**
- Menu de navegação fixo com efeitos de scroll
- Navegação mobile com menu hambúrguer
- Scroll suave entre seções
- Indicadores visuais de posição

### 3. **Seções Principais**
- **Hero Section**: Apresentação impactante com call-to-action
- **Sobre**: Informações sobre a empresa e diferenciais
- **Produtos**: Catálogo de produtos com modal de detalhes
- **Serviços**: Explicação do conceito "Pegue e Monte"
- **Contato**: Formulário de contato e informações

### 4. **Interatividade**
- Animações de scroll
- Efeitos hover em cards
- Modal de produtos com informações detalhadas
- Validação de formulário em tempo real
- Notificações de sucesso/erro

### 5. **Sistema de Produtos**
- Catálogo com categorias (Decoração, Mesa, Iluminação, Tintas)
- Modal com detalhes de produtos
- Sistema de pacotes de locação
- Cores e variações disponíveis

### 6. **Formulário de Contato**
- Validação de email
- Formatação automática de telefone
- Envio simulado com feedback visual
- Mensagens de sucesso/notificação
